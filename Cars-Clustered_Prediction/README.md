# UAS Machine Learning - Used Cars

Project ini bertujuan untuk melakukan analisis dan prediksi kategori mobil bekas menggunakan metode machine learning.

## Metode yang digunakan
- Clustering menggunakan K-Means
- Klasifikasi menggunakan Random Forest

## Fitur Sistem
- Input data kendaraan melalui web
- Prediksi kategori mobil secara real-time
- Dashboard hasil prediksi

## Teknologi
- Python
- Scikit-learn
- Flask

## Cara menjalankan
1. Install dependencies:
   pip install -r requirements.txt

2. Jalankan aplikasi:
   python run.py

3. Buka browser:
   http://localhost:5000
